## 🎯 Sobre o Projeto

Este projeto é um sistema de controle de estoque desenvolvido com foco em aprendizado prático e evolução para um produto real.

A aplicação permite gerenciar produtos e quantidades de forma simples, simulando cenários comuns de sistemas empresariais.

> 💡 Este projeto faz parte da minha jornada de transição para desenvolvimento Full Stack.

---

## 🧠 Problema que resolve

Pequenos negócios muitas vezes não possuem ferramentas simples para controle de estoque.

Este sistema foi pensado para:

* organizar produtos
* controlar quantidades
* facilitar o gerenciamento básico

---

## ⚙️ Funcionalidades

* ✅ Cadastro de produtos
* ✏️ Edição de itens
* ❌ Remoção de produtos
* 📊 Controle de quantidade
* 💾 Persistência com localStorage
